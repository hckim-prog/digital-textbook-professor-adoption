# 한빛+ 디지털 교재 교수 홍보 페이지

## 프로젝트 목적

한빛아카데미 교재를 강의에 채택하거나 검토하는 대학교수를 대상으로, 한빛+ 디지털 교재의 수업 활용 방식과 학생 학습 지원 기능을 안내하는 정적 랜딩페이지입니다.

## 실행 방법

별도 빌드 과정 없이 정적 파일로 실행할 수 있습니다.

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

브라우저에서 `http://127.0.0.1:4173/index.html`을 열어 확인합니다.

## 기존 저자 안내 사이트와의 차이

기준 사이트인 `digital-textbook-guide`는 저자에게 디지털 교재 전환과 확인 절차를 안내하는 페이지였습니다. 이 프로젝트는 기존 디자인, 레이아웃, 이미지, 영상 자산을 유지하되 내용을 일반 대학교수 대상 안내형 홍보 페이지로 바꾸었습니다.

수익, 계약, 저자 확인 절차가 아니라 강의 준비 부담 완화, 학생 예습과 복습, 부가 학습 자료 활용, 교재 채택 검토를 중심으로 설명합니다.

## 수정한 주요 섹션

- 히어로 영역을 교수 대상 디지털 교재 소개 문구로 변경
- 1분 시연 영상 안내 문구를 수업 활용 관점으로 변경
- 통합 뷰어 소개를 교수와 학생 모두의 학습 환경 관점으로 수정
- 강의교안, 쉬운 해설, 해설 영상, 추가 연습문제, AI 튜터 설명을 강의 운영과 학생 복습 중심으로 변경
- 기존 디지털 교재 제작 흐름을 강의 채택 및 활용 흐름으로 재구성
- 정산 관련 섹션을 교수와 학생의 장점 섹션으로 교체
- 체험 영역을 샘플 교재 체험 관점으로 수정
- FAQ 전체를 일반 교수 대상 질문과 답변으로 교체
- 마지막 문의 영역을 강의 채택 검토와 샘플 체험 문의로 정리

## Vercel 배포 주소

예상 주소: https://digital-textbook-promo.vercel.app/

현재 로컬 Vercel CLI에 저장된 토큰이 유효하지 않으면 직접 배포가 실패할 수 있습니다. 이 경우 아래 순서로 배포합니다.

1. Vercel CLI에 다시 로그인합니다.

```powershell
vercel login
```

2. 현재 프로젝트 폴더에서 GitHub 저장소와 같은 이름의 Vercel 프로젝트로 연결합니다.

```powershell
vercel link --yes --project digital-textbook-promo
```

3. 정적 사이트를 프로덕션으로 배포합니다.

```powershell
vercel --prod
```

4. 배포가 끝나면 Vercel이 출력한 실제 URL을 이 README의 배포 주소에 기록합니다.
